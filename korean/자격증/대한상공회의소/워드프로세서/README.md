
# 워드프로세서

## 종목소개

기업에서 다량의 문서처리가 이루어지면서
빠르고 정확한 문서작성이 요구됨.

워드프로세서 검정은 컴퓨터의 기초사용법과
효율적인 문서작성을 위한
워드프로세서 프로그램 운영 및 편집능력을 평가하는
국가기술자격 시험.
